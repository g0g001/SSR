
安装环境

    Debian 8.2 


shell安装

one-ssr.sh

在线一键安装

 wget -qO one-ssr.sh http://suo.im/17OB7R && bash one-ssr.sh

uninstall.sh

wget -qO uninstall.sh http://suo.im/3hKMAS && bash uninstall.sh
